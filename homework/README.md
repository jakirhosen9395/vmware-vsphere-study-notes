# Homework

This directory contains homework assignments and a submission template for structured learning.

## Structure

- **Assignments**: Practical exercises to apply your knowledge
- **Submission Template**: Standardized format for documenting your work

## Assignment Types

Homework may include:
- Research assignments
- Lab exercises
- Configuration tasks
- Troubleshooting scenarios
- Documentation exercises

## Submission Template

Use the `submission-template.md` file to document your homework completion:
- Assignment details
- Steps taken
- Screenshots/evidence
- Challenges encountered
- Lessons learned

## Available Assignments

Add your assignment files here. Examples:
- `assignment-01-lab-planning.md`
- `assignment-02-esxi-deployment.md`
- `assignment-03-vm-lifecycle.md`
- `submission-template.md` - Template for submitting homework
