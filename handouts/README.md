# Handouts

This directory contains quick reference materials including cheatsheets and diagrams.

## Contents

### Cheatsheets
Quick reference guides for common tasks and commands:
- ESXi commands
- PowerCLI cmdlets
- Keyboard shortcuts
- Configuration settings
- Best practices

### Diagrams
Visual representations of vSphere concepts (in Mermaid format .mmd):
- Architecture diagrams
- Network topology examples
- Storage configurations
- HA/DRS workflows
- vMotion processes

## File Formats

- **Markdown (.md)**: For cheatsheets and text-based references
- **Mermaid (.mmd)**: For diagrams that can be rendered as visuals
- **Images (.png, .jpg)**: For screenshots and complex diagrams

## Usage

These handouts are designed for:
- Quick lookups during labs
- Study aids for certification
- Real-world reference during admin tasks
- Team knowledge sharing
