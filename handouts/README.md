# Handouts

This directory contains quick reference materials for VMware vSphere.

## Contents

### Cheatsheets
Quick reference guides for common tasks and commands.

### Diagrams
Architecture diagrams and visual references in Mermaid (.mmd) format.

## Using Mermaid Diagrams
Mermaid diagrams can be viewed in:
- GitHub (native support)
- VS Code (with Mermaid extension)
- Online editors (mermaid.live)
