# VMware vSphere Glossary

A comprehensive glossary of VMware vSphere terms and concepts.

---

## A

**Affinity Rule**  
A rule that keeps virtual machines together on the same host.

**Anti-Affinity Rule**  
A rule that keeps virtual machines separated on different hosts.

---

## D

**Datastore**  
A logical storage container that holds virtual machine files, templates, and ISO images.

**DRS (Distributed Resource Scheduler)**  
A feature that automatically balances computing workloads with available resources.

**DPM (Distributed Power Management)**  
A feature that optimizes power consumption by placing hosts in standby mode during low utilization.

---

## E

**ESXi**  
VMware's bare-metal hypervisor that directly installs on physical servers.

**ESXCLI**  
A command-line interface for managing ESXi hosts.

---

## H

**HA (High Availability)**  
A feature that provides automatic failover for virtual machines in case of host failure.

**Hypervisor**  
Software that creates and runs virtual machines.

---

## N

**NFS (Network File System)**  
A network storage protocol supported by vSphere.

**NIC (Network Interface Card)**  
A physical network adapter on a host.

---

## P

**PowerCLI**  
VMware's PowerShell-based command-line interface for automation.

**PSOD (Purple Screen of Death)**  
ESXi's equivalent of a system crash screen.

---

## R

**Resource Pool**  
A logical abstraction for flexible management of CPU and memory resources.

---

## S

**Snapshot**  
A point-in-time capture of a virtual machine's state and data.

**Storage vMotion**  
Live migration of virtual machine files from one datastore to another.

---

## T

**Template**  
A master copy of a virtual machine used to create new VMs.

---

## V

**vCenter Server**  
Centralized management platform for VMware vSphere environments.

**VLAN (Virtual Local Area Network)**  
A logical network segment within a physical network.

**VM (Virtual Machine)**  
A software-based computer that runs on a physical host.

**VMDK (Virtual Machine Disk)**  
The file format for virtual machine hard disks.

**VMkernel**  
The ESXi kernel that manages system resources.

**VMFS (Virtual Machine File System)**  
VMware's clustered file system for storing virtual machines.

**vMotion**  
Live migration of running virtual machines from one host to another without downtime.

**vSAN (Virtual SAN)**  
VMware's software-defined storage solution.

**vSphere**  
VMware's virtualization platform consisting of ESXi and vCenter Server.

**vSwitch (Virtual Switch)**  
A software-based network switch within ESXi.

---

## W

**WWPN (World Wide Port Name)**  
A unique identifier for Fibre Channel ports.

**WWNN (World Wide Node Name)**  
A unique identifier for Fibre Channel nodes.
