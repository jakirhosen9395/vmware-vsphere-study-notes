# Automation Scripts

This directory contains automation scripts for VMware vSphere management.

## Script Types

### Bash Scripts
Scripts for Linux/macOS environments for:
- ESXi management
- Automation tasks
- Monitoring

### PowerShell Scripts
PowerCLI scripts for:
- vSphere automation
- Bulk operations
- Reporting

## Usage
Each script includes:
- Purpose and description
- Prerequisites
- Usage instructions
- Examples

## Safety
Always test scripts in a lab environment before using in production.
