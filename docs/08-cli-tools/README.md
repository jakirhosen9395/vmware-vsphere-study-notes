# Module 08: CLI Tools

This module introduces command-line tools for vSphere management.

## Topics Covered
- ESXi Shell (Local Tech Support Mode)
- SSH access to ESXi
- ESXCLI command-line interface
- PowerCLI (PowerShell for VMware)
- govc (vSphere CLI written in Go)
- Common administrative tasks via CLI
- Scripting and automation basics
