# Module 06: HA, DRS, and vMotion

This module covers vSphere's core availability and resource management features.

## Topics Covered
- vSphere High Availability (HA)
  - HA prerequisites and configuration
  - Admission control policies
  - VM and application monitoring
- vSphere DRS (Distributed Resource Scheduler)
  - DRS automation levels
  - Affinity and anti-affinity rules
  - Resource pools and shares
- vMotion
  - Live migration concepts
  - vMotion requirements
  - Storage vMotion
  - Enhanced vMotion Compatibility (EVC)
