# Quizzes

This directory contains module quizzes to test your knowledge and an answer key for self-assessment.

## Structure

- **Module Quizzes**: One quiz per study module to reinforce learning
- **Answer Key**: Solutions and explanations for all quiz questions

## Quiz Format

Each quiz includes:
- Multiple choice questions
- True/False questions
- Scenario-based questions
- Practice questions similar to certification exams

## How to Use

1. Complete the corresponding study module first
2. Take the quiz without looking at answers
3. Check your answers against the answer key
4. Review any topics where you struggled
5. Retake the quiz until you can score 90% or higher

## Available Quizzes

Add your quiz files here. Examples:
- `quiz-00-intro.md`
- `quiz-01-virtualization.md`
- `quiz-02-vmware-basics.md`
- `answer-key.md` - Contains all quiz answers
