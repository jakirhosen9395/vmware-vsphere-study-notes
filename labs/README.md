# Labs

This directory contains step-by-step lab guides for hands-on practice with VMware vSphere.

## Available Labs

Labs are organized to complement the study modules and provide practical experience.

### Lab Structure
Each lab includes:
- **Objective**: What you'll learn
- **Prerequisites**: Required knowledge and resources
- **Step-by-step instructions**: Detailed walkthrough
- **Verification**: How to confirm successful completion
- **Troubleshooting**: Common issues and solutions

## Getting Started

1. Ensure your home lab environment is set up (see `docs/03-lab-setup`)
2. Start with foundational labs before attempting advanced ones
3. Take notes on any issues you encounter
4. Use the troubleshooting guide if needed
